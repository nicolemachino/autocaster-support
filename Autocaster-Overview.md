Vue d’ensemble Autocaster = API Node.js/TypeScript + UI React/Vite servant une réalisation automatique TriCaster (WebSocket audio/tally + HTTP macros/framebuffers + API Trigger In en GET). Données persistantes : apps/api/data/system.json, apps/api/data/users.json, apps/api/data/shows/*.json, logs dans apps/api/logs/ (fichier courant autocaster-YYYY-MM-DDTHH-MM-SSZ.log). Version : GIT_COMMIT pour l’API, VITE_GIT_DESCRIBE ou VITE_GIT_COMMIT injecté au build UI (sinon dev). Modes Autocam : ON, OFF (stop), PAUSE, FREEZE (gel temporaire), SPECIAL (via scenarios IN/OUT).
